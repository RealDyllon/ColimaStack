# ColimaStack Command API Reference

This repository-level note points to the maintained Starlight page:

- `docs/src/content/docs/reference/command-api.md`

The public reference is source-verified against the current Swift services and documents the exact command shapes ColimaStack invokes for:

- Colima profile lifecycle and diagnostics
- Docker inventory and monitor data
- Kubernetes inventory and metrics
- tool lookup and forwarded environment
- local Colima file paths

Keep this file short so command examples do not drift in two places.
