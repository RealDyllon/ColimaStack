---
title: Menu Bar App
description: Use ColimaStack from the macOS menu bar.
---

ColimaStack includes a macOS menu bar surface for quick runtime access.

Use the menu bar when you need to check profile state or return to the main workspace without navigating through the full app first.

The menu bar should remain a lightweight entry point. Detailed diagnostics, Docker resources, Kubernetes resources, logs, and configuration belong in the main window.
